# Calculator

## 概要

- アプリ開発の練習を目的とした Android 向けのシンプルな電卓アプリ
- Google Play ストアへの公開を目標にプライベートリポジトリで開発を始めていたが、そこまで力を入れるのは難しい気がしてきたので公開することにした。
